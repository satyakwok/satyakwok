Rust developer. I build open-source libraries, developer tools, and blockchain infrastructure — in public.

Right now I'm building **[Sentrix Chain](https://github.com/sentrix-labs/sentrix)**, an open-source EVM Layer-1 in Rust, and maintaining **[reliakit](https://github.com/satyakwok/reliakit)**, a `no_std` reliability toolkit for Rust. I also contribute upstream to the Rust and Ethereum ecosystem — including a merged fix in [Alloy](https://github.com/alloy-rs/alloy).
